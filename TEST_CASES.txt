Test 3.1 - Decode valid token

  [Golden Token]
Expected Header:
  {
    "alg": "HS256",
    "typ": "JWT"
  }
Expected Payload:
  {
    "sub": "1234567890",
    "name": "John Doe",
    "iat": 1516239022
  }

  No error message

Test 3.2 - Verify valid signature

  [Golden Token]

  [Golden Secret]

  Signature Verified

  [Golden Token]

  wrong-secret

  Invalid Signature

    "alg": "HS256",
